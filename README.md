# Página de Pokémons

Bem-vindo à Página de Pokémons! Este projeto é uma página web desenvolvida com HTML, CSS e JavaScript, apresentando informações sobre diversos Pokémons.

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Como Visualizar

Baixe ou clone o repositório em seu computador.
Abra o arquivo `index.html` em seu navegador web.

## Recursos

- **Lista de Pokémons**: Apresenta uma lista de Pokémons com suas imagens e informações.
- **Design Temático e Atraente**: Interface inspirada no universo Pokémon.

## Estrutura do Projeto

- `index.html`: Contém a estrutura HTML da página principal.
- `style.css`: Arquivo de estilo CSS para estilizar a página.
- `script.js`: Script JavaScript responsável pela lógica interativa.

## Personalização

- Modifique o arquivo `index.html` para ajustar a estrutura conforme necessário.
- Adapte o estilo no arquivo `style.css` para atender às suas preferências.
- Explore o arquivo `script.js` para personalizar a lógica ou adicionar mais funcionalidades.

## Pré-requisitos

Certifique-se de ter um navegador web instalado para visualizar o projeto.
